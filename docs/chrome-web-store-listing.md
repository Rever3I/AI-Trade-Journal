## Detailed Description

### 中文介绍

**AI Trade Journal — 你的智能交易复盘助手**

告别手动整理交易记录。AI Trade Journal 是一款专为中国日内交易者打造的 Chrome 扩展，帮你将券商交易数据一键同步到 Notion，并由 AI 自动生成专业的交易复盘分析。

**核心功能：**

**Smart Paste 智能粘贴**
直接粘贴券商导出的交易记录（CSV、制表符分隔或自由文本格式），AI 自动识别并解析为结构化数据。每条解析结果附带置信度评分，让你清楚了解数据准确性。解析完成后一键预览、确认并同步到你的 Notion 交易日志。

**支持 5+ 主流券商**
- 富途 (Futu) — 完整支持中文 CSV 导出
- 盈透证券 (IBKR) — 支持 Flex Query 报告
- 微牛 (Webull) — 支持标准导出格式
- 嘉信理财 (Schwab) — 支持 CSV 导出
- 老虎证券 / 长桥 (Tiger / Longbridge) — 支持标准导出

同时支持自由文本输入，即使是手动输入的交易记录也能智能识别。

**6 大 AI 分析模板**
由专业交易教练级 AI 为你提供深度分析，每次分析控制在 200 字以内，精炼实用：

1. **日内复盘** — 每日交易总结，包含盈亏统计、情绪状态评估和改进建议
2. **单笔深挖** — 选择单笔交易进行深度分析，包含进出场评估和假设推演
3. **周度统计** — 每周胜率趋势、最佳策略和风险指标汇总
4. **策略评估** — 评估特定策略在一段时间内的表现和有效性
5. **情绪体检** — 识别报复性交易、FOMO、过度交易等行为模式
6. **风险体检** — 仓位管理、相关性分析和最大回撤评估

**Notion 深度整合**
- 通过 OAuth 安全连接你的 Notion 工作空间
- 自动创建专业交易日志数据库（附赠精心设计的模板）
- 每笔交易自动记录：代码、方向、数量、价格、盈亏、手续费
- AI 分析结果自动填入对应交易条目
- 支持情绪标签、策略分类、AI 评分等高级字段

**定价说明**
- 一次性购买 ¥299，终身使用
- 无需订阅 Notion AI（我们自带 AI 分析能力）
- 无需编写任何提示词，开箱即用
- 每日赠送免费 AI 分析额度，也可使用自己的 Claude API Key

**隐私与安全**
- 交易数据通过安全的服务端代理处理，绝不在客户端暴露 API 密钥

### English Description

**AI Trade Journal — Your Intelligent Trading Review Assistant**

Stop manually organizing trade records. AI Trade Journal is a Chrome extension built for Chinese retail day traders that syncs broker trade data to Notion with AI-powered professional trade review analysis.

**Key Features:**

- **Smart Paste:** Paste broker exports (CSV, tab-separated, or free text) and AI automatically parses them into structured data with confidence scoring. Preview, confirm, and sync to Notion in one click.

- **5+ Broker Support:** Futu, IBKR (Interactive Brokers), Webull, Schwab, Tiger/Longbridge — plus free-text input for any format.

- **6 AI Analysis Templates:**
  1. Daily Review — end-of-day summary with P/L, emotional state, and improvement suggestions
  2. Single Trade Deep-Dive — entry/exit analysis with what-if scenarios
  3. Weekly Stats — win rate trends, best setups, risk metrics
  4. Strategy Evaluation — specific strategy performance over time
  5. Emotion Check — behavioral pattern detection (revenge trading, FOMO, overtrading)
  6. Risk Assessment — position sizing, correlation, max drawdown analysis

- **Notion Integration:** OAuth connection, automatic database creation with a professional template, structured trade entries with AI analysis, emotion tags, strategy classification, and AI scoring.

**Pricing:**
- One-time purchase at 299 CNY, lifetime access
- No Notion AI subscription required
- No prompt writing needed
- Daily free AI analysis quota included; option to use your own Claude API Key

**Privacy:**
- Trade data processed via secure server-side proxy
- No browsing history, personal identity, or financial credentials collected
- No data sold to third parties
